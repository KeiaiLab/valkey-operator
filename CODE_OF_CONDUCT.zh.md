# Contributor Covenant 行为准则

## 我们的承诺

作为成员、贡献者和领导者,我们承诺让所有人参与本社区的体验都不受骚扰,无论年龄、体型、可见或不可见的残疾、族裔、性别特征、性别认同与表达、经验程度、教育程度、社会经济地位、国籍、外貌、种族、宗教、性认同与性取向如何。

我们承诺以有助于建设开放、包容、多元、健康社区的方式行动与互动。

## 我们的标准

有助于为本社区营造积极环境的行为示例:

* 对他人展现同理心与善意 (empathy and kindness)
* 尊重不同的观点、看法与经验
* 给予并优雅地接受建设性反馈 (constructive feedback)
* 对自身错误承担责任并向受影响者致歉,从经历中学习
* 关注的不仅是个人,而是整个社区的最佳利益

不可接受的行为示例:

* 使用性化语言或图像,以及任何形式的性关注或性挑逗
* 喷子行为 (trolling)、侮辱或贬低性评论,以及人身或政治攻击
* 公开或私下的骚扰
* 未经明确许可,公开他人的隐私信息,例如物理地址或邮箱地址
* 在专业场合中可被合理判定为不当的其他行为

## 我们的责任

项目维护者负责澄清并执行我们的行为标准,并将对任何不可接受的行为做出恰当与公正的纠正措施。

项目维护者有权利与责任移除、编辑或拒绝不符合本行为准则的评论、commit、代码、wiki 编辑、issue 与其他贡献,并在适当时沟通做出该 moderation 决定的原因。

## 适用范围

本行为准则适用于所有社区空间,并且在个人以官方身份代表社区出现在公开场合时同样适用。代表社区的示例包括:使用官方电子邮箱、通过官方社交账号发布内容,或在线上 / 线下活动中担任指定代表。

## 执行

针对辱骂、骚扰或其他不可接受行为的事件,可以通过下方列出的渠道向项目维护者报告。所有投诉都将得到及时、公正的审查与调查。

所有项目维护者均有义务尊重事件报告人的隐私与安全。

## 执行指南

不遵守本行为准则的项目维护者可能会面临由项目领导层其他成员决定的临时或永久后果。

## 出处

本行为准则改编自 [Contributor Covenant][homepage] 版本 2.1,网址: [https://www.contributor-covenant.org/version/2/1/code_of_conduct.html][v2.1]。

Community Impact Guidelines 灵感来自 [Mozilla 的行为准则执行阶梯][mozilla coc]。

如有关于本行为准则的常见问题,请参阅 [https://www.contributor-covenant.org/faq][faq] 的 FAQ。翻译版本请见 [https://www.contributor-covenant.org/translations][translations]。

## 报告指南

如果您经历或目睹了不可接受的行为,或有任何其他顾虑,请通过以下方式联系项目维护者进行报告:

* **GitHub Security Advisory** (推荐用于敏感报告): https://github.com/keiailab/valkey-operator/security/advisories/new
* **Email**: `security@keiailab.com` (PGP key 见 `artifacthub-repo.yml`)

所有报告都将以审慎与保密的方式处理。我们会尽快回应您的报告并采取适当措施。
