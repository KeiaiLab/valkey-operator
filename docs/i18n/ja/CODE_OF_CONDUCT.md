# Contributor Covenant 行動規範

## 私たちの誓約

私たちはメンバー、コントリビューター、リーダーとして、年齢、体型、目に見える / 見えない障害、エスニシティ、性別の特徴、ジェンダーアイデンティティおよびその表現、経験のレベル、教育、社会経済的地位、国籍、外見、人種、宗教、性的アイデンティティおよび性的指向にかかわらず、私たちのコミュニティへの参加をハラスメントのない体験とすることを誓約します。

私たちは、開かれた、歓迎的で、多様性を尊重し、包括的で、健全なコミュニティに資する形で行動し、相互に交流することを誓約します。

## 私たちの基準

私たちのコミュニティにとって肯定的な環境に寄与する行動の例:

* 他者への共感と思いやりを示すこと
* 異なる意見、視点、経験を尊重すること
* 建設的なフィードバックを与え、また快く受け入れること
* 自らの過ちに対して責任を受け入れ、影響を受けた人々に謝罪し、その経験から学ぶこと
* 個人としてだけでなく、コミュニティ全体にとって何が最善かに焦点を当てること

容認できない行動の例:

* 性的な表現や画像の使用、およびあらゆる種類の性的注目や誘い
* トローリング、侮辱的または軽蔑的なコメント、個人的または政治的な攻撃
* 公の場または私的な場でのハラスメント
* 物理的住所やメールアドレスなど、他者の私的情報を明示的な許可なく公開すること
* 専門的な環境において不適切とみなされ得るその他の行為

## 私たちの責任

プロジェクトメンテナーは、容認可能な行動の基準を明確化し執行する責任を負い、容認できない行動の事例に対して適切かつ公正な是正措置を講じます。

プロジェクトメンテナーは、本行動規範に整合しないコメント、コミット、コード、wiki 編集、issue、その他のコントリビューションを削除、編集、または却下する権利と責任を有し、適切な場合にはモデレーション判断の理由を伝達します。

## 適用範囲

本行動規範は、すべてのコミュニティスペース内で適用され、個人が公的な場でコミュニティを公式に代表している場合にも適用されます。コミュニティ代表の例には、公式メールアドレスの使用、公式 SNS アカウントを介した投稿、オンラインまたはオフラインのイベントで指名された代表者として行動することが含まれます。

## 執行

虐待的、ハラスメント的、その他容認できない行動の事例は、以下に指定されたチャネルを通じてプロジェクトメンテナーに報告できます。すべての苦情は速やかかつ公正にレビューおよび調査されます。

すべてのプロジェクトメンテナーは、いかなる事案においても報告者のプライバシーとセキュリティを尊重する義務があります。

## 執行ガイドライン

本行動規範を遵守しないプロジェクトメンテナーは、プロジェクトリーダーシップの他のメンバーが判断した一時的または恒久的な措置を受ける可能性があります。

## 帰属

本行動規範は [Contributor Covenant][homepage] バージョン 2.1 ([https://www.contributor-covenant.org/version/2/1/code_of_conduct.html][v2.1]) を翻案したものです。

Community Impact Guidelines は [Mozilla の行動規範執行ラダー][mozilla coc] にインスパイアされています。

本行動規範に関する一般的な質問への回答は [https://www.contributor-covenant.org/faq][faq] の FAQ を参照してください。翻訳は [https://www.contributor-covenant.org/translations][translations] で入手できます。

## 報告ガイドライン

容認できない行動を体験または目撃した場合、あるいはその他懸念がある場合は、以下のチャネルでプロジェクトメンテナーに報告してください:

* **GitHub Security Advisory** (機密性の高い報告に推奨): https://github.com/keiailab/valkey-operator/security/advisories/new
* **メール**: `security@keiailab.com` (PGP key は `artifacthub-repo.yml` に記載)

すべての報告は慎重かつ機密性をもって取り扱われます。可能な限り迅速にご報告に対応し、適切な措置を講じます。
