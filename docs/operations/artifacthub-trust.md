## Applying Signed Status

Artifact Hub displays the Helm `Signed` state only when
`<chart>-<version>.tgz.prov` is available next to the chart archive. Helm
provenance is generated by `helm package --sign`.

Default release command:

```bash
make release VERSION=vX.Y.Z
```

The `Makefile` defaults to `HELM_SIGN=1`, so the command fails unless the
following prerequisites are present:

- `~/.gnupg/secring.gpg`
- `HELM_GPG_KEY=Keiailab Helm`
- `HELM_GPG_FINGERPRINT=89A409476828CB992338C378651E51AF520BCB78`

Preparing the secret key on a new development machine:

```bash
gpg --import <keiailab-helm-private-key.asc>
gpg --export-secret-keys > ~/.gnupg/secring.gpg
make helm-signing-preflight
```

Post-release verification:

```bash
bash scripts/release-smoke-test.sh vX.Y.Z
```

The smoke test verifies the `.tgz.prov` asset in GitHub Releases, the `.tgz.prov`
file on `gh-pages`, and `helm verify` using the public signing key from
`artifacthub-repo.yml`.

## Applying Official Status

`Official` cannot be enabled by repository files alone. In Artifact Hub, it is
an externally reviewed state indicating that the publisher owns the software the
package primarily focuses on.

Satisfied prerequisites:

- Artifact Hub repository ID:
  `16085dd0-0f19-4c6b-ab90-bd97105bdf42`
- Verified publisher: `true`
- Chart README: `charts/valkey-operator/README.md`
- Repository metadata served:
  `https://keiailab.github.io/valkey-operator/artifacthub-repo.yml`

Remaining steps:

1. An Artifact Hub publisher or organization member opens an official status
   request issue in `artifacthub/hub`.
2. Request repository-level official status.
3. Include the following facts in the request.

```text
Repository name: keiailab-valkey-operator
Package name: valkey-operator
Repository URL: https://keiailab.github.io/valkey-operator
Package URL: https://artifacthub.io/packages/helm/keiailab-valkey-operator/valkey-operator
Publisher owns the software: yes, keiailab publishes and maintains valkey-operator itself.
Verified publisher: true
Documentation: charts/valkey-operator/README.md
```

Until Artifact Hub review is complete, local code changes or `gh-pages` publish
operations cannot change the `Official` badge to `true`.
